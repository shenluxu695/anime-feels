Production Pipeline

End-to-end automated anime short video creation in 6 steps.

1

AI Scriptwriting

DeepSeek generates emotional anime story scripts with scene-by-scene breakdown, dialogue, and visual prompts.

Powered by DeepSeek V4 Flash
2

Scene Generation

AI creates anime-style images for each scene based on detailed prompts, maintaining character consistency.

Powered by Luma Photon + Character Reference
3

Voiceover Narration

Emotional voice dubbing with multiple character voices, synced to script dialogue timing.

Powered by ElevenLabs TTS
4

Background Music

Custom BGM generated from real instrument SoundFont libraries, matched to scene emotion.

Powered by FluidSynth + FluidR3_GM
5

Video Compilation

FFmpeg assembles scenes, audio, subtitles, and transitions into a polished short-form video.

FFmpeg + ASS Subtitles + Loudnorm Audio
6

Publish to TikTok

Directly upload and publish videos to TikTok via Content Posting API with customizable captions and settings.

TikTok Content Posting API (Direct Post)
