TikTok Login Integration

Anime Feels uses TikTok's Login Kit to securely connect your TikTok account with our platform. Below is how the authorization flow works.

Authorization Flow

1
User clicks "Connect TikTok Account" on Anime Feels platform
2
TikTok authorization page opens via OAuth 2.0 redirect
3
User reviews requested permissions: profile info, video upload, video list
4
User authorizes the app — TikTok returns an authorization code
5
Anime Feels exchanges the code for an access token via TikTok API
6
Account connected! User can now publish videos directly from our platform

Requested Scopes

  • user.info.basic — Read profile info (open_id, avatar, display name)
  • user.info.profile — Read profile link, bio description
  • user.info.stats — Read user stats (followers, likes, video count)
  • video.list — Read user's public videos
  • video.upload — Upload and publish videos to TikTok

OAuth Configuration

Redirect URI for the OAuth callback:

https://shenluxu695.github.io/anime-feels/callback

This endpoint receives the authorization code and exchanges it for an access token using the TikTok API.
