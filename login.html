TikTok Login Integration

Anime Feels uses TikTok Login Kit for secure OAuth 2.0 authorization.

Authorization Flow

1
User clicks "Connect TikTok Account" in Anime Feels dashboard
2
TikTok OAuth dialog opens, requesting user authorization
3
User reviews scopes and clicks Authorize
4
TikTok redirects to our callback endpoint with authorization code
5
Server exchanges code for access token via TikTok API
6
Account connected! Dashboard shows TikTok stats and publishing controls

Requested Permissions (Scopes)

  • user.info.basic Read profile info (open_id, avatar, display name) — Login Kit
  • user.info.profile Read profile link and bio description
  • user.info.stats Read follower count, likes count, video count
  • video.list Read user's public video list
  • video.upload Upload and publish videos directly to TikTok — Content Posting API

OAuth Configuration

Redirect URI (where TikTok returns the authorization code):

https://shenluxu695.github.io/anime-feels/callback

The server-side handler exchanges the code for an access token at:

POST https://open.tiktokapis.com/v2/oauth/token/
