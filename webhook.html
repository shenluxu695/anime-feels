TikTok Webhook Integration

Anime Feels receives real-time event notifications from TikTok via webhooks. These events allow the platform to stay synchronized with user activity.

Webhook Events

  • Video Status Updates — Notifications when uploaded videos are processed, published, or encounter errors
  • Authorization Changes — Alerts when users revoke or modify app permissions
  • Content Compliance — Updates regarding content policy reviews and status changes

Configuration

Webhook Callback URL:

https://shenluxu695.github.io/anime-feels/webhook

This endpoint receives POST requests from TikTok with JSON payloads containing event data. The server validates the webhook signature before processing.

Webhook Payload Format

Example webhook event:

{
  "event": "video.status",
  "timestamp": 1714352000,
  "data": {
    "publish_id": "abc123",
    "status": "PUBLISH_COMPLETE",
    "video_id": "741234567890"
  }
}
