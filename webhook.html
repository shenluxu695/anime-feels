TikTok Webhook Integration

Real-time event notifications from TikTok to keep Anime Feels synchronized.

Webhook Endpoint

https://shenluxu695.github.io/anime-feels/webhook

This URL receives POST requests from TikTok containing JSON event payloads. The server validates the HMAC signature before processing.

Supported Events

  • Video Status Updates — Notifications when uploaded videos are processed, published, or fail
  • Authorization Changes — Alerts when users revoke or modify app permissions
  • Content Compliance — Updates on content policy review results

Sample Payload

{
  "event": "video.status",
  "timestamp": 1714352000,
  "data": {
    "publish_id": "abc123",
    "status": "PUBLISH_COMPLETE",
    "video_id": "741234567890"
  }
}
